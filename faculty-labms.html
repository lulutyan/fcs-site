Лаборатория микропроцессорных систем

Кафедры Вычислительных Систем

Заведующий кафедрой

Коробицын Виктор Викторович - к.ф.-м.н., доцент

Лаборанты

Нихматулина Альбина Темирбаевна

Шеянова Ирина Михайловна

Лаборатория располагается в двух смежных аудиториях и предназначена для проведения различных лабораторных работ по дисциплинам, связанным с аппаратным обеспечением компьютерных систем.

Техническое обеспечение:

  • Компьютеры на базе Intel Pentium IV (12 шт)
  • Учебный микропроцессорный комплекс УМПК-51
  • Учебный микропроцессорный комплекс УМПК-80
  • Аппаратно-программный комплекс на базе процессоров Intel MCS-196 и TMS320C2X
  • Программаторы AMTEL STK-500 (6 шт)
  • Программаторы AMTEL STK-501, STK-502
  • Учебный комплекс по схемотехнике УМ-11
  • Монтажный стол, оборудованный паяльной станцией
  • Комплект оборудования для беспроводной связи Bluetooth и WiFi
  • Осциллографы (2 шт)
  • Генератор Г3-118
  • Мультиметры ВС-21А (3 шт)
  • Компьютерные стенды для изучения периферийных устройств (3 шт)
  • Ноутбук, лазерный принтер, струйный принтер, сканер
  • Набор компьютерных комплектующих для лабораторных работ

Программное обеспечение:

  • ОС Windows XP
  • Пакет программ Open Office
  • Пакет для разработки микропроцессорных систем AVR Studio
  • Пакет анализа электронных схем 5 Space Analysis
  • Геоинформационная система Quantum GIS
  • Геоинформационная система Дубль-ГИС

Поддерживаемые курсы:

    по cпециальности 230101 - вычислительные машины, комплексы, системы и сети
  • Организация ЭВМ и систем
  • Схемотехника ЭВМ
  • Микропроцессорные системы
  • Сервисное обслуживание вычислительной техники
  • Разработка ГИС
  • Интерфейсы периферийных устройств
    по специальности 090102 - компьютерная безопасность
  • Аппаратные средства вычислительной техники
  • Схемотехника
